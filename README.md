# wow-recipes
This is my first project from the Foundations course of The Odin Project! The goal is to create a basic recipe website using HTML.

I'll be using recipes from World of Warcraft: The Official Cookbook by Chelsea Monroe-Cassel.

## Skills Demonstrated
- Use Git and GitHub to track and commit changes
- Create files and directories using the command line
- Set up basic HTML structure
- Use headings, paragraphs, and lists
- Link pages and embed images using relative paths
